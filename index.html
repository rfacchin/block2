PROJECT OVERVIEW

The “space project” is a backend (server-side) solution for a space agency to control their astronauts, mission, targets, and attends. With the use of the correct technology, this application will cover all the daily tasks.

The order to insert / create a entry in the system is to follow the order tabs from 1 to 4.

It is possible to edit each entry (foreign key dependencies should be followed) by clicking on the requiered tab and "MANAGE BUTTON"

Due the dependency (foreign key ) the order to DELETE must be from tab 4 to 1.
